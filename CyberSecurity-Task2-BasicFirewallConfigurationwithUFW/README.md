## what does a firewall do?
-A firewall is designed to inspect a network or a digital device's incoming and outgoing traffic.
-The goal is the same as that of a security guard which is not letting any unathorized visitor enter a system or a network.

## what does each firewall rule achive?
The rules are customized to allow or block specific traffic

## Why did i choose these specific rules?
sudo ufw allow ssh(it allows SSH traffic on port 22)
sudo ufw deny http it denies any HTTP traffic on port 80
